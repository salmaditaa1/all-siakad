# Sistem Informasi Akademik (SIAKAD)

## Deskripsi

Aplikasi Sistem Informasi Akademik berbasis web untuk mengelola data fakultas dan program studi.

## Tech Stack

### Frontend

* Next.js
* React

### Backend

* FastAPI

### Database

* PostgreSQL

## Features

* Login
* Register
* Manajemen Fakultas
* Manajemen Program Studi

## Cara Menjalankan

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
